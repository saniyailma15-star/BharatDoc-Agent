# 🇮🇳 BharatDoc Agent

### AI-Powered Multilingual Document Intelligence System

BharatDoc Agent is an AI-powered platform that transforms unstructured documents into intelligent digital records through OCR, structured data extraction, document analysis, and conversational querying.

---

## Problem Statement

India generates millions of multilingual documents every day. Manual document verification, data entry, and information retrieval are time-consuming and error-prone.

BharatDoc Agent addresses this challenge by digitizing documents and converting them into structured, searchable, and actionable information.

---

## Key Features

### 📄 OCR-Based Document Processing

* Extracts text from uploaded documents
* Supports image-based documents

### 🌐 Multilingual Support

* English
* Hindi
* Tamil
* Kannada
* Telugu
* Malayalam
* Marathi

### 📑 Document Type Detection

Automatically identifies:

* Aadhaar Card
* PAN Card
* Passport
* Certificate
* Educational Documents
* Invoices
* Student Documents

### 📋 Structured Data Extraction

Converts raw text into structured key-value pairs.

### 🛡 Reliability Assessment

Evaluates document confidence and quality.

### 🚨 Document Risk Analyzer

* Detects missing fields
* Calculates document health score
* Flags potentially suspicious documents

### 📊 Analytics Dashboard

Displays:

* OCR Confidence
* Extracted Fields
* Text Lines
* Language Information

### 🔍 Search Inside Document

Allows users to search for keywords within extracted content.

### 💬 Chat With Document

Ask questions about uploaded documents and receive intelligent responses.

### 📦 JSON Export

Exports processed document data in machine-readable JSON format.

### 🤖 AI Summary

Automatically generates document insights and summaries.
