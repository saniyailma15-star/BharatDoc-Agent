# 🇮🇳 BharatDoc Agent

## AI-Powered Multilingual Document Intelligence System

### Problem Statement

Government and institutional documents are often stored as scanned images or paper records, making information retrieval, analysis, and digitization difficult.

### Solution

BharatDoc Agent is an AI-powered document intelligence platform that extracts, understands, and structures information from multilingual documents using OCR and intelligent document analysis.

### Features

* OCR-based text extraction
* Multilingual support

  * English
  * Hindi
  * Tamil
  * Kannada
  * Telugu
  * Malayalam
  * Marathi
* Document type detection
* Structured field extraction
* Analytics dashboard
* AI-generated document summary
* Chat with Document
* JSON export and download

### Technology Stack

* Python
* Streamlit
* PaddleOCR
* Pillow
* JSON

### Workflow

1. Upload document
2. OCR extracts text
3. Document type is detected
4. Key fields are extracted
5. Analytics are generated
6. AI summary is created
7. User interacts using Chat with Document
8. Structured JSON is exported
